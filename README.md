# Prototipo de Termómetro Clínico

Este es un proyecto experimental para tratar de crear un termómetro a distancia de forma compacta y precisa. En vez de usar una pantalla, se ha preferido usar
el móvil como pantalla y panel de control a través de una sencilla aplicación creada en MIT App Inventor, para ello se ha establecido una comunicación Bluetooth 
de Bajas Emisiones.
